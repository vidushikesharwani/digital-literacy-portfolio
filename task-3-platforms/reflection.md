# Task 3 — Reflection

## Platforms Used

### HackerRank
HackerRank is an online coding practice platform used 
by students and professionals worldwide to improve 
programming and problem-solving skills. I completed 
the "Solve Me First" beginner challenge using Java, 
which required writing a function to return the sum 
of two integers. Although the problem was simple, it 
helped me get comfortable with HackerRank's coding 
interface and submission system.

### Google Forms
Google Forms is a free cloud-based tool for creating 
surveys, quizzes, and feedback forms. I created a 
5-question Digital Literacy Awareness Quiz covering 
topics like password safety, phishing, and digital 
tools. I linked the form responses to a Google Sheet 
for automatic data collection.

## What I Built
- A completed HackerRank coding challenge using Java
- A Digital Literacy Awareness Quiz with MCQ and 
  short answer questions
- A linked Google Sheet to collect and view responses

## How These Tools Will Help Me Academically
HackerRank will help me prepare for technical 
interviews and improve my Java and problem-solving 
skills. Google Forms will be useful for conducting 
surveys and collecting peer feedback for academic 
projects. Both platforms are industry-relevant tools 
that I will continue using throughout my degree.
