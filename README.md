##  Project Overview
This repository contains my Digital Literacy Portfolio 
developed as part of the Digital Literacy course. The 
goal of this project is to build practical knowledge of 
essential digital skill: from creating awareness 
resources and setting up a professional online presence, 
to practising responsible communication and understanding 
cybercrime prevention.

As a Student Digital Ambassador, this portfolio reflects 
my commitment to helping peers navigate the digital world 
safely, professionally and effectively.

##  Objectives
- Understand the concept and importance of digital literacy
- Build a professional and credible online presence
- Explore coding practice and cloud collaboration platforms
- Learn and apply proper email etiquette
